Listas ordenadas

Existem alguns tipos de listas ordenadas, que mudam a nomecaltura das sequencias numéricas utilizadas nas listas. e elas são:

Tipos de listas

  • I - numerais romanos em maiúculas
  • i - numerais romanos em minúsculas
  • 1 - valor padrão, cria uma lista numerada
  • A - lista alfabética maiúscula
  • a - lista alfabética minúscula
    •

Para inserir estes tipos utilizamos o parâmetro "type" dentro da tag de abertura <ol type="I">

  3. Tomar café
  4. Escovar os dentes

Tipo a

  1. Acordar
  2. Ligar para o João
  3. Tomar café

Outra alteração possivel é selecionar o numero pelo qual a lista se dará início, nada impede que sua lista começe pelo numero 3, pelo numero 20 ou até pelo numero 5000.

Para faze-lo utilize o parâmetro "start" dentro da tag de abertura, ex:
<ol type"1" start "5">
