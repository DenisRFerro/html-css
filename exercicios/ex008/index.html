Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase temos um termo em destaque usando a <strong> </strong> (semântica).

Italíco / enfase

Nesta frase temos um termo em itálico usando a tag I (não semântica).

Nesta frase temos um termo em enfase usadno uma tag em <em> </em> (semântica),

As tag's B e I ainda estão em uso porem o recomendado e deixar de usar por pertencerem a uma versão
mais antiga do html, por isso existe a tendencia de cair em desuso.

Sempre se atentar também as questoe de Semântica e forma, HTML e CSS respectivamente,

Texto marcado

Podemos criar também um texto marcado usando a tag <mark> </mark>.

Podemos modificar a cor da tag mark usando o comando da CSS <style> para mudar uma tag especifica.

Para mudar a cor de todas as tag <mark> presentes no texto posso adicionar a tag <style> na"head" do meu
codigo, fazendo assim com que todas as tag adicionadas em seguida sigam a cor selecionada nesta linha de
codigo.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

A tag <big> pode cair em desuso.

Já a tag <small> é usado para criar um texto pequeno.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

Esta marcação é feita a partir da tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

utilizando a tag <ins>.

Existe também o sublinhado com a tag <u> (não semântica).

Mesmo tendo o mesmo aspecto a tag <u> tende a cair em desuso.

Texto sobrescrito

Utiliza-se a tag <sup>

Para inserir coisas do tipo: x20+3

Texto subscrito

Utiliza-se a tag <sub>

Para inserir coisas do tipo: H2o
